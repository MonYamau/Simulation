# Simulation

Консольное приложение "simulation" демонстрирует пошаговую симуляцию места, населённого кошками и мышами. Задача кошек 
выловить всех вредителей на карте, а мышей — съесть весь отыскиваемый сыр.
Проект выполнен с целью практики использования принципов ООП на Java 21.

## Инструкция

1. Предварительно установите в терминале кодировку UTF-8 для корректного отображения объектов.
2. Запустите приложение.
3. Введите в консоль букву кириллического алфавита (допустимы как строчные, так и заглавные):
"Х" — выполнить один ход;
"Н" — запустить/продолжить бесконечную симуляцию;
"П" — приостановить бесконечную симуляцию;
"В" — выйти из приложения.
4. Отслеживайте прогресс симуляции на отображаемом интерфейсе.
